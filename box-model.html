Conceito de Box Model

O Box Model no CSS define como os elementos da página são representados no layout. Cada elemento é envolvido em um "box" que inclui as seguintes propriedades:

  • Content (Conteúdo): A área onde o texto ou outro conteúdo é exibido.
  • Padding (Preenchimento): Espaço entre o conteúdo e a borda.
  • Border (Borda): A linha ao redor do conteúdo e do padding.
  • Margin (Margem): Espaço externo ao redor da borda.
